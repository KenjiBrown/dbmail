NAME

dbmail.conf - configuration file for DBMail Message Delivery Agent.

SYNOPSIS
--------
There are 8 sections: DBMAIL, SMTP, LMTP, POP, IMAP, SIEVE, LDAP, DELIVERY.
The default config file has documentation for all settings.
